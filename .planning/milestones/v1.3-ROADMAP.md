# Milestone v1.3: MCP Sampling Debugging

**Status:** ✅ SHIPPED 2026-04-13
**Phases:** 15-16
**Total Plans:** 2

## Overview

本 milestone 聚焦在现有单 server MCP 工作台基础上补齐 `sampling` 调试能力，并确保它自然融入现有请求构造、结果查看、history / replay / diagnostics 体系，而不是变成一条割裂的新入口。

## Phases

### Phase 15: Sampling Request Flow

**Goal**: 把 `sampling` 接入现有单 server MCP 工作台主链路
**Depends on**: None
**Plans**: 1 plan

Plans:

- [x] 15-PLAN: Sampling Request Flow

**Details:**
- 在 MCP workbench 中支持 `sampling` 请求主链路
- 提供结构化输入、边界提示、可读响应呈现
- 保持与现有 workbench 一致，而不是引入平行工作流

### Phase 16: Replay, Diagnostics, and Fit

**Goal**: 让 `sampling` 融入现有 history / replay / diagnostics 与状态流
**Depends on**: Phase 15
**Plans**: 1 plan

Plans:

- [x] 16-PLAN: Replay, Diagnostics, and Fit

**Details:**
- `sampling` 进入现有 history / replay 闭环
- 历史列表保持紧凑摘要，但底层保留回放保真
