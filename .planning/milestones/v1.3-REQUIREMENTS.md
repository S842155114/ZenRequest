# Requirements Archive: ZenRequest v1.3

**Archived:** 2026-04-13
**Core Value:** 让开发者以本地优先、快速、可控的方式完成高频 API 调试与工作流操作，而不被臃肿云平台和账号绑定打断。

## v1.3 Requirements

### MCP Sampling

- [x] **MCPS-01**: 开发者可以调试 MCP `sampling` 能力，并检查其输入输出与安全边界
- [x] **MCPS-02**: 开发者可以在工作台中查看 `sampling` 请求的结构化结果与诊断信息
- [x] **MCPS-03**: 开发者可以像其他 MCP 操作一样保留 `sampling` 的 history / replay 体验

### Workbench Integration

- [x] **WBIN-01**: `sampling` 应集成到现有单 server MCP 工作台，而不是引入独立工作流界面
- [x] **WBIN-02**: `sampling` 交互应遵循当前 MCP 工作台的状态、错误处理与展示模式
