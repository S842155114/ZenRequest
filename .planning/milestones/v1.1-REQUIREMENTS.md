# Requirements Archive: ZenRequest v1.1

**Archived:** 2026-04-10
**Core Value:** 让开发者以本地优先、快速、可控的方式完成高频 API 调试与工作流操作，而不被臃肿云平台和账号绑定打断。

## Completed Requirements

### MCP Resources

- [x] **MCPR-01**: 开发者可以列出 MCP server 暴露的 resources
- [x] **MCPR-02**: 开发者可以读取单个 MCP resource，并查看结构化结果与原始协议内容
- [x] **MCPR-03**: 开发者可以回放 resource 读取请求，并保留足够的上下文用于诊断

### MCP Prompts

- [x] **MCPP-01**: 开发者可以列出 MCP server 暴露的 prompts
- [x] **MCPP-02**: 开发者可以为 prompt 输入参数并执行 prompt 获取结果
- [x] **MCPP-03**: 开发者可以在 prompt 调试过程中查看结构化结果、原始协议包与历史摘要

### MCP Roots

- [x] **MCPROOT-01**: 开发者可以为当前 MCP 调试会话配置 roots 输入
- [x] **MCPROOT-02**: 开发者可以在 roots 相关请求中看到 roots 配置已实际传递给 MCP server

### MCP Transport

- [x] **MCPT-01**: 开发者可以通过 `stdio` 连接并调试单个本地 MCP server
- [x] **MCPT-02**: 开发者可以在 HTTP 与 `stdio` 两种传输方式下获得一致的工作台主链路体验
- [x] **MCPT-03**: 开发者在 `stdio` MCP 调试失败时可以看到结构化且可定位的错误信息

## Outcome Notes

- All 11 v1.1 requirements were delivered and verified during phases 08–11.
- Milestone audit passed in `.planning/v1.1-MILESTONE-AUDIT.md`.
